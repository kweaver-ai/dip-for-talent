# KWeaver 智能人才分析解决方案（简要说明）

[English](README.md) | 中文

KWeaver 面向中大型组织的人力资源与业务管理场景，强调数据可信、逻辑可靠与决策可解释，融合多源人才与业务数据，形成统一的人才分析与辅助决策能力，把复杂分散的人才问题转化为可判断、可行动的管理决策闭环。

## 方案构成（设计文档摘要）
- **核心模块**：人才洞察工作台、人力资本 ROI 分析、人岗动态匹配分析、组织健康度分析、核心人才激励有效性分析、人员流失风险分析，以及行动中心、数据中心、配置中心等支撑模块。
- **ROI 指标基线**：ROI 以“产出收益与投入成本”衡量效率，作为关键管理指标之一。

## ROI Analysis（roi-analysis 设计文档摘要）
- **业务目标**：建立“人力 → 业务结果”的量化关系，支持组织层级下钻分析，为组织调整、激励策略与资源再分配提供依据。
- **典型场景**：月度/季度经营复盘、区域/部门绩效对比、组织调整前评估、激励或编制调整量化决策。
- **核心功能**：
  - ROI 核心指标总览：指标卡展示当前值、基准值、达成率及同比趋势，并支持查看口径与趋势。
  - 组织架构 ROI 分布：以组织树展示不同层级的 ROI 表现，支持对比与下钻。
  - 人效关联指标分析：展示与 ROI 强相关指标、相关系数、正负相关方向及影响解读。
  - Copilot 交互：围绕 ROI 指标与趋势、根因与行动建议进行解释与问答。
